Portfolio Tomka

Nowoczesne portfolio deweloperskie zbudowane w Next.js, TypeScript i Tailwind CSS. Łączy czystą prezentację treści, płynne animacje oraz zoptymalizowane doświadczenie użytkownika. Prezentuje profil zawodowy, projekty, doświadczenie, wykształcenie i kontakt w przejrzystej, responsywnej formie.

Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

Features
- Responsywny layout zoptymalizowany pod mobile, tablet i desktop
- Płynne animacje UI z Framer Motion
- SEO-friendly metadata i Open Graph
- Integracja z GitHub heatmap do wizualizacji aktywności
- Modularne sekcje: About, Projects, Experience, Education, Contact
- Zoptymalizowane ładowanie zasobów
- Czysta, minimalistyczna szata graficzna

Setup
npm install
npm run dev

Tomek's Portfolio

A modern developer portfolio built with Next.js, TypeScript, and Tailwind CSS. Combines clean content presentation, smooth motion, and an optimized user experience. Showcases professional profile, projects, experience, education, and contact in a clear, responsive layout.

Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

- Vercel
